OsManager

Sistema moderno de gerenciamento de Ordens de Serviço desenvolvido para portfólio.

Stack:

• FastAPI
• React
• TypeScript
• PostgreSQL
• SQLAlchemy
• Docker
• JWT

Objetivo

Construir um sistema completo de gerenciamento de ordens de serviço utilizando boas práticas de arquitetura, autenticação, banco de dados e desenvolvimento full stack.
